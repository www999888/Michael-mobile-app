## 🎯 核心功能

1. **实时盯盘** - 实时监控股票价格、涨跌幅、成交量
2. **技术分析** - MA、MACD、KDJ、RSI、BOLL
3. **AI 分析** - Ollama 本地大模型生成交易建议
4. **模拟交易** - 模拟买卖、持仓管理、盈亏统计
5. **告警系统** - 价格告警、技术指标告警

## 📝 使用说明

### 启动应用
```bash
cd ai-stock-monitor
run.bat
```
访问 http://localhost:8501

### 自定义股票池
编辑 `config/settings.py`:
```python
STOCK_POOLS = {
    "沪深 A 股": ["000001.SZ", "600000.SH"],
}
```

### 选择 AI 模型
支持模型：
- `qwen3.5:35b` (推荐)
- `glm-4.7-flash`
- `llama4`

在 UI 侧边栏选择模型。

### 调整告警阈值
编辑 `config/settings.py`:
```python
ALERT_THRESHOLD = {
    "price_change_pct": 5.0,   # 涨跌幅告警阈值
    "rsi_overbought": 70,      # RSI 超买阈值
}
```

### 模拟交易配置
编辑 `config/settings.py`:
```python
SIM_TRADING = {
    "initial_capital": 1000000,  # 初始资金
    "stop_loss_pct": 5.0,        # 止损比例
    "take_profit_pct": 10.0,     # 止盈比例
}
```
