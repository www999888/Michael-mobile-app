**数据模块** (`data/`):

- ✅ 支持东方财富接口 / Tushare
- ✅ 模拟数据支持（开发测试用）

**分析模块** (`analyzer/`):
- ✅ `TechAnalyzer` - 技术指标计算

  - MACD（指数平滑异同）
  - KDJ（随机指标）
  - RSI（相对强弱指标）
  - BOLL（布林带）
- ✅ `AIAnalyzer` - 本地 AI 智能分析
  - Ollama 集成
  - 交易建议生成
  - 置信度评估

**盯盘模块** (`monitor/`):
- ✅ `StockMonitor` - 实时监控
- ✅ 价格涨跌幅告警
- ✅ 技术指标告警
- ✅ 桌面通知支持

## 🚀 快速开始

### 1. 安装依赖
```bash
cd ai-stock-monitor
pip install -r requirements.txt
```

### 2. 启动应用
```bash
run.bat
# 或
streamlit run main.py
```

### 3. 访问界面
浏览器打开：http://localhost:8501

### 4. 打包 EXE（可选）
```bash
build_exe.bat
# 生成：dist\AI_Stock_Monitor.exe
```
